Jump Reset for Gravity Boots

Difficulty★★★★★

Utility/Commonly Used★★★★★

When you find Gravity Boots without Leap Stone, you’ll notice that you can’t chain Gravity Boots. This is because in order to do chain gravity jumps, you need to be in the double jump air state. If you don’t have double jump, you can access this air state by transforming and untransforming mid-air (any transformation) or using a thrust sword special mid-air and then chaining gravity jumps. This can be very useful to access some flight-required checks that you would otherwise not be able to access. Check Understanding Air States for more info.
