Colosseum with only an Air Reset

Difficulty★★★★★

Utility/Commonly Used★★★★★

Similarly to the trick above, you can reset your air state by using a transformation or thrust sword to access the double jump state. This state allows you to dive kick, which can be used to bounce on top of candles for extra height. In particular, if you go to the clock room you will notice there is a candle below the path to the Colosseum/Olrox Quarters. If you jump, then above the candle you reset your state with a thrust sword or transformation, you can divekick into the candle to gain extra height and access this route.
