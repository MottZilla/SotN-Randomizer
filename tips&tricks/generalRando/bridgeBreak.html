Bridge Break

Difficulty★★★★★

Utility/Commonly Used★★★★★

Bridge Break is a hidden option that allows you to access the Mines/Catacombs without having to use the explosive barrel to open the bridge. Exclusive to the PSX version, if a player presses triangle on their 2nd controller when in the bridge room, the bridge will open directly. For those using the Launcher, this button comes already set up to L3/LS (pressing down the left stick).
